# The Referee – Beginner Python Project

This project is created for the Kiro Hero Challenge as part of the AI for Bharat program.

## Description
The Referee is a simple Python program that compares two options instead of giving a single answer.
It explains pros, cons, and gives a recommendation to help users make better decisions.

## Comparisons included
- AWS EC2 vs AWS Lambda
- Python vs Java

## How to run
1. Install Python
2. Run the file using:
   python referee.py
